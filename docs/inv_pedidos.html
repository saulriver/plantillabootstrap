Pedidos

Este módulo no es una transacción de SISCOS, es un preámbulo a la entrega pedidos no mueve inventarios. Requisitos Bodega, Área, Personal y Periodo activo etc. Para trabajar este módulo se debe haber configurado el stock máximo del técnico para que se pueda ejercer el control es decir el stock personal.

Para este caso se agrega material de movistar o material propio, no se tienen que hacer pedido para CT o EECC. En el mismo pedido se puede combinar todo, el pedido va amarrado al área, las áreas si no se pueden combinar.

El usuario debe dar clic en el módulo Pedidos y se visualizará la siguiente ventana que requiere de información para realizar el registro.
Se escoge el Departamento y la Bodega con que se va a trabajar para realizar el pedido.
Fecha entrega: Es muy importante para cuando va ir el técnico a la bodega a reclamar el pedido.
Entrega a: Técnico que se le va a entregar ese material.

Nueva entrega

Esta función se encuentra ubicada en la barra vertical lado izquierdo, se debe dar clic en la opción Nueva entrega cuando ya se ingresa toda la información solicitada u obligatoria para que se activen los demás botones y se realice el proceso de hacer el pedido.

Buscar personal

Esta función se encuentra ubicada en la barra horizontal superior lado izquierdo, permite al usuario poder hacer la búsqueda con comodines, para que se visualice todo el personal registrado en el sistema SISCOS y seleccionar el necesario para el campo entrega a ese pedido que se le va a realizar.
Comodines de búsquedas tales como.
  • (%) Todos.
  • (%ADRIAN) Los que terminan.
  • (ADRIAN%) Los que comiencen.

Stock personal

Esta opción se encuentra ubicada en la barra horizontal superior, se debe dar clic en la opción Stock personal y este permite al usuario ver que tiene el técnico cargado actualmente cuál es su stock actual, arrojará toda la información de este técnico.

El usuario también visualizará una pestaña de materiales y seriales que se pueden mandar a Excel en el botón Generar informe Excel que se encuentra ubicado en la barra horizontal inferior.

Agregar material

Esta función se encuentra ubicada en la barra horizontal superior, se debe dar clic en la opción Agregar material que es lo que se le va a entregar al técnico.

Si el usuario no tiene la información de los materiales que quiere agregar al pedido para esto tiene la opción Buscar que al dar clic sobre él se visualizará otra ventana donde se requiere de comodines de búsquedas tales como.
  • (%) Todos.
  • (%CABLE) Los que terminan.
  • (CABLE%) Los que comiencen.
Seguidamente se analizan todos los materiales registrados en el sistema como se puede ver a continuación.
Posteriormente el usuario se ubica en la celda Código y dar clic, automáticamente se traslada la información a la pantalla de Stock disponibles materiales. Si el material le existe la reserva se hace posible el traslado de lo contrario mostrará mensaje de error al usuario que el producto no tiene existencia.
Luego de la pantalla mencionada anteriormente stock disponibles materiales, el usuario se ubica en la celda Reserva y da clic en la tecla Enter y la información automáticamente se traslada a la pantalla principal de pedidos y los materiales han sido agregados correctamente.
Aquí no se escogen seriales el que los determina es Almacén, cuando ya se tengan todos los materiales se les pone la cantidad en Existencia actualmente.

Nota: El usuario debe tener claro que materiales va agregar que pertenezcan a la misma área que fue seleccionada anteriormente de agregar material, porque de lo contrario presentará mensaje de error al querer agregar y debe ingresar la celda obligatoria  Cantidad para poder realizar el registro.

Eliminar material

Esta función se encuentra ubicada en la barra horizontal superior, permite al usuario eliminar un material que se haya registrado en el listado de pedidos y que no crea conveniente para este proceso.

Registrar entrega

Esta opción se encuentra ubicada en la barra vertical superior lado izquierdo, permitiendo al usuario cuando ya tiene toda la información solicitada u obligatoria dar clic sobre en la opción Registrar entrega para guardarla en el sistema.
El sistema mostrará un mensaje de información el cual se le debe dar clic en el icono Sí para que se efectué el registro de pedido.
Con este ticket es que va el técnico a la bodega, el pedido no imprime ningún formato.

Salir

Para salir del módulo es a través del botón Salir que se encuentra ubicado en la barra horizontal superior, dar clic.
Posteriormente se debe dar clic en el icono Sí, para finalizar el proceso de Pedidos.
