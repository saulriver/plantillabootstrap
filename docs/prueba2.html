Nuestros Productos

Gescob

Software Gestión de Operativa de Cobros

GESCOB ™ es una aplicación Web con un componente móvil que permite monitorear las campañas de gestión de cobros a través de dispositivos móviles como Smartphones, Tablets.

Odisea

Software gestión de actividades

Odisea ™ es una aplicación web que permite administrar las diferentes actividades con sus respectivos recursos. De una manera centralizada y Online ODISEA™ permite gestionar cada una de las actividades asignadas.

Sici

Software Control de Inventarios

SiCi es un Software que le permite a las empresas controlar los inventarios de una manera rápida y fácil. Es completamente configurable y adaptable a los diferentes entornos de acuerdo a las necesidades del cliente.

Ticket Manager

Gestión de Tickets (eventos)

TICKET MANAGER ™ es una plataforma web que le permite administrar las actividades o eventos que les generan sus clientes, estos eventos pueden estar asociados a Proyectos, Tipo de trabajo y unos ANS (tiempos de resolución).

Recaudasoft

Plataforma Recaudo

RECAUDASOFT ™ es un software que permite llevar el control a los diferentes puntos de recaudo para el pago de servicios públicos. Realiza interfaz con los diferentes operadores a través de WebServices o con el archivo de ASOBANCARIA 1998, 2001 y 2010.

Sofatec

Software administración Técnicos

SOFATEC ™ es una aplicación Web y Móvil que permite administrar las actividades de los técnicos a través de un dispositivo móvil (Smartphone), el sistema permite cargar las actividades, asignarlas a los técnicos y hacerles seguimiento en tiempo real.
